# Privacy Policy (Template)

**Effective date:** TBD

TraceMacApp is a local macOS trace viewer for Terra/OpenTelemetry trace files.

## Local-only by default

By default, TraceMacApp:

- Reads trace files from a folder on your Mac.
- Does not automatically upload traces anywhere.

## Network access

TraceMacApp may access the network only when you explicitly enable or initiate features such as:

- **App updates** (direct distribution, via Sparkle)
- **Crash reporting** (if enabled)

## Trace content

Trace files can contain metadata about model/agent/tool execution. The instrumented app controls what is written (Terra defaults are privacy-safe; prompt capture is opt-in).

## Diagnostics export

When you choose “Export Diagnostics…”, TraceMacApp creates a zip you can share with support. It may include:

- app version + OS version
- current settings (folder watch enabled, etc.)
- a listing of trace files (names + sizes + timestamps)
- an app log file (high-level events like folder selection and activation attempts)

You control where this file is saved and who you share it with.
